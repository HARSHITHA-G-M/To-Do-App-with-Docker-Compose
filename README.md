## ⚙️ Getting Started

### 1️⃣ Prerequisites

Make sure you have:

- Docker 🐳 installed → https://docs.docker.com/get-docker/
- Docker Compose 🧩 installed → included with Docker Desktop

---

### 2️⃣ Run the App 🚀
# Clone this repo
git clone 
cd todo-app

# Start all services
docker-compose up --build
